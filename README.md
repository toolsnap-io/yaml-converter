# YAML to JSON Converter

A free online tool to convert between YAML and JSON formats instantly.

## Features

- Convert YAML to JSON
- Convert JSON to YAML
- Syntax highlighting for both formats
- Error detection with line number reporting
- Pretty-print and minified output options
- Support for complex nested structures

## How to Use

1. Paste your YAML or JSON data in the editor
2. Select the target format (YAML → JSON or JSON → YAML)
3. Click "Convert" to see the result
4. Copy or download the converted output

## Supported Formats

- **YAML 1.2** — Full spec support including anchors, aliases, and multi-line strings
- **JSON** — Standard JSON with validation
